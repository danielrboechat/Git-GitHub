# :octocat: Git and GitHub Course

This repository contains my **notes files** from the **Git and GitHub** course by **Gustavo Guanabara**, available on the [**Curso em Vídeo**](https://www.cursoemvideo.com/) platform.

The course has a total workload of **approximately 20 hours** and focuses on teaching the fundamentals of **version control** and how to use Git and GitHub in real projects.

## What I learned in this course

- What Git is and why version control matters
- What GitHub is and how it is used in real projects
- The history and evolution of Git and GitHub
- Creating and managing repositories
- Using GitHub Desktop
- Cloning repositories
- Versioning existing projects
- Working with GitHub Issues
- Using Markdown for documentation
- Basic GitHub security practices
- Working with branches
- Hosting websites for free using GitHub Pages

## Purpose of this repository

- Learn the basics of Git and GitHub  
- Practice version control in real scenarios  
- Understand collaborative workflows  
- Build a solid foundation for future projects
